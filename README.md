### 1. Большая часть грамматики языка
Файл MLL.g4 предсталяющий грамматику находится в *src/MLL/*

Пример программы на созданном языке:

```
const int a = 0;
const int b = 0.3;
const int c = a + b;

def funcName(){
    print(a + b);
}

main(){
    int g = 34;
    int h = funcName(g) + 5;
    print(h);
    print(3, 4);
}
```
